# Adfix Web

React + Vite frontend for Adfix PM.

## Run
1. Ensure API is running at `http://localhost:4000`.
2. Install workspace dependencies from repo root: `npm install`.
3. Start web app from repo root: `npm run dev:web`.

## Environment
- `VITE_API_BASE_URL` (optional): defaults to `http://localhost:4000/api/v1`.
